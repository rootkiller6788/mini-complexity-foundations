# Polynomial Hierarchy

**MIT 6.840 §8 · AB §5 · Stockmeyer (1976)**

Sigma_0 = P, Sigma_{k+1} = NP^{Sigma_k}, Pi_k = co-Sigma_k.
PH = union over all k.

## Key Facts
- P=NP => PH=P
- BPP subset Sigma_2 (Sipser-Gacs-Lautemann)
- Open: is PH infinite?
