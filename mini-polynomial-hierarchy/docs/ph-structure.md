# Polynomial Hierarchy Structure

## Definition
Sigma_k^p = NP^{Sigma_{k-1}^p}, Pi_k^p = co-Sigma_k^p
PH = Union of all Sigma_k^p

## Collapse
If Sigma_k^p = Pi_k^p then PH collapses to level k.

## Key Theorems
- Karp-Lipton: NP in P/poly => PH = Sigma_2
- Toda: PH subset P^{#P}
- Sipser-Gacs-Lautemann: BPP subset Sigma_2
