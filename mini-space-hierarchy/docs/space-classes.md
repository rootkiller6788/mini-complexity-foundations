# Space Complexity Classes

## L (Logspace)
SPACE(log n). O(log n) work tape, read-only input.

## NL (Nondeterministic Logspace)  
NSPACE(log n). Complete: ST-CONNECTIVITY.

## Key Results
- Savitch: NL subset SPACE(log^2 n)
- Immerman-Szelepcsenyi: NL = coNL (1988)
- Space Hierarchy: SPACE(f) < SPACE(g) for f=o(g)
