# Major Space Complexity Theorems

## Savitch (1970): NSPACE(s) ⊆ SPACE(s²)
- Recursive reachability algorithm
- Implies PSPACE = NPSPACE
- Space = O(s²) for path testing

## Immerman-Szelepcsenyi (1987): NL = coNL
- Inductive counting technique
- First non-trivial collapse of nondeterministic space class
- Godel Prize 1995

## Chandra-Kozen-Stockmeyer (1981): AP = PSPACE
- Alternating Turing Machines
- ATIME(poly) = PSPACE, ALOGSPACE = P

## Reingold (2008): Undirected connectivity ∈ L
- Zig-zag product of expander graphs
- Proves SL = L (symmetric logspace = deterministic logspace)
