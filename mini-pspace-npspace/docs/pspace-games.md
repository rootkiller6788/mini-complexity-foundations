# PSPACE-Complete Games

## Known PSPACE-Complete Games
- Generalized Geography
- Hex (arbitrary board)
- Checkers (n x n board) — Fraenkel et al. 1978
- Go (Japanese rules) — Lichtenstein-Sipser 1980
- Othello/Reversi — Iwata-Kasai 1994
- Rush Hour — Flake-Baum 2002
- Sokoban — Culberson 1997

## Common Structure
All these games share: 2-player alternation, poly(n) moves,
state described in poly(n) bits. Game tree search = PSPACE.
