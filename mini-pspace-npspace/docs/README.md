# PSPACE & NPSPACE

**AB §4 · MIT 6.840 §8 · Savitch (1970) · Immerman-Szelepcsenyi (1987)**

## Space Complexity Hierarchy
L ⊆ NL ⊆ P ⊆ NP ⊆ PSPACE = NPSPACE ⊆ EXP

## Key Theorems
- Savitch: NSPACE(s) ⊆ SPACE(s²) → PSPACE = NPSPACE
- Immerman-Szelepcsenyi: NL = coNL
- TQBF is PSPACE-complete (analogous to SAT for NP)
- AP = PSPACE (Chandra-Kozen-Stockmeyer)

## Open Problems
L vs NL · NL vs P · P vs PSPACE · PSPACE vs EXP
