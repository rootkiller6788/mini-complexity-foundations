# Course Tree: mini-reductions-completeness

## Prerequisites (Depends On)
- Basic combinatorics and graph theory
- Big-O notation and asymptotic analysis
- Basic logic (propositional logic, CNF)
- Turing machines (from mini-cook-levin-theorem)

## This Module Covers
- Polynomial-time reductions (many-one, Turing, logspace)
- NP-completeness definition and properties
- Karp's reduction chain (21 problems)
- Gadget construction methodology
- Closure properties under reductions
- Completeness beyond NP (PSPACE, EXP, PH)

## Feeds Into (Dependents)
- mini-p-np-np-completeness (full NP-completeness theory)
- mini-cook-levin-theorem (detailed Cook-Levin)
- mini-ladner-theorem (NP-intermediate)
- mini-polynomial-hierarchy (PH completeness)
- mini-pspace-npspace (PSPACE completeness)
- mini-circuit-complexity (circuit reductions)
- mini-approximation-algorithms (hardness of approx)
- mini-pcp-theorem (PCP and gap reductions)
- mini-fine-grained-complexity (fine-grained reductions)

## Learning Path
1. Start here: Basic reductions (this module)
2. Then: mini-cook-levin-theorem (deeper proof)
3. Then: mini-p-np-np-completeness (full theory)
4. Advanced: mini-polynomial-hierarchy, mini-pspace-npspace
5. Further: mini-circuit-complexity, mini-pcp-theorem
